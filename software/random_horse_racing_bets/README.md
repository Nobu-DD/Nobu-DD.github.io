# 競馬買い目ランダム出力機能
## 概要
このミニアプリは予想する時間が無い...だけど競馬はやりたいと思った方に向けた、完全自動買い目出力アプリになります。<br>
馬の戦績や状態を全く関与せず、完全に運に乗っかった出力をするので、予想するのがバカバカしくなった方にもおすすめです。
## アプリを作ったキッカケ
私自身、普段のプログラミング学習や、乗馬・麻雀を趣味として始めた代償として、競馬予想に充てる時間が無くなってしまいました。一旦競馬をやるのは控えようかと思いましたが、穴馬を本命にして馬券に絡む快感を定期的に味わいたい...<br>
というわけで、手軽に競馬に参戦出来るアプリを作ることがキッカケの一つになりました。<br>
あとは純粋にJSのキャッチアップの足がかりとして着手いたしました。
